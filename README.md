# vue-custom-scrollview

基于 better-scroll 的一个可自定义上下拉的移动端滚动列表组件。
上下拉现默认动画，如果你想做成自己想要的上下拉动画，请查看[api](https://www.yuque.com/xiaotianyi/lkig5m/af04bx)
### Use Setup
```
npm i vue-custom-scrollview -S
```
